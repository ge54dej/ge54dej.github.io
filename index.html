Bachelor Thesis

Towards Automated Chemistry Research: Making Laboratory Devices Intelligent and Automation-Compatible

Demo
Automated experiment cycle: robot loads a plate, the device makes the experiment and the robot unloads it.
Motivation

Lab workflows often require manual handling and one-off device integrations. This slows experiments and increases errors.

Approach

Used SiLA 2 software as a common interface so robots and lab devices can be orchestrated in one workflow.

Result

Demonstrated a complete end-to-end run and a structure that can be extended to other devices and workflows.

Tools
Franka Panda SiLA 2 Python C++

DrawBot — Drawing shapes from SVG on a whiteboard

Built a drawing pipeline in RoboDK to render SVG shapes on a whiteboard with a UR10e.

Key points
  • Converted SVG paths into drawing trajectories (strokes + pen up/down moves).
  • Scaled and positioned shapes consistently on the board.
  • Validated on multiple drawings (spaceship, circle, gear).

PolishBot — Erasing a drawing from the surface

Implemented an erase routine by teaching a wipe trajectory and replaying it reliably.

Key points
